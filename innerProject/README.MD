# Vue Demo

* 安装nodejs、npm；
* 命令行运行 npm install；
* 命令行运行 npm run start；
* 浏览器默认 localhost:8080


#已安装配置

*element.ui

*axios(引入一个封装好的js文件)

*json-server（虚拟本地服务器，来获取请求数据）

*less安装与配置

*引入重置css文件样式


#规范（demo里设置的文件放置地方）
